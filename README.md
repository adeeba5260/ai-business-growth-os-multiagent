## Overview
AI-powered real-time marketing assistant using Streamlit, Groq API, and LLMs for strategy generation and business insights.

---

## Features
- AI-generated marketing strategies
- Multi-agent workflow
- Prompt engineering
- Business insights generation
- Structured JSON output parsing
- Growth prediction and execution planning

---

## Tech Stack
- Python
- Streamlit
- Groq API
- LLMs
- JSON
- Regex

---

## Installation

```bash
pip install -r requirements.txt
python -m streamlit run ui.py
```

---

## Project Workflow

1. User enters business or marketing requirements
2. AI agents process the request
3. Groq API generates AI responses
4. Strategies and insights are analyzed
5. Results are displayed using Streamlit

---
