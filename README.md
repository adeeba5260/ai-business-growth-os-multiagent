AI-powered real-time marketing assistant using Streamlit, Groq API, and LLMs for strategy generation and KPI dashboards.

- Multi-agent workflow
- KPI dashboards
- Prompt engineering
- Business insights generation
- Structured JSON output parsing

## Tech Stack
- Python
- Streamlit
- Groq API
- LLMs
- JSON
- Regex
